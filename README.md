## ToneBlock

A sliding block puzzle with sounds. [Play it online here.][ToneBlock]

Built with [Solid][] and [Elementary][].

To install:

```bash
pnpm install
```

To run:

```bash
pnpm dev
```

To build:

```bash
pnpm build
```

[ToneBlock]: https://toneblock.erikostrom.com/
[Solid]: https://www.solidjs.com/
[Elementary]: https://www.elementary.audio/
